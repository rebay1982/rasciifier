# rASCIIfier 
rASCIIfier - A simple tool to convert PNG images to ASCII art.

## Description

Born out of curiousity about processing images and generating ASCII art from there, here is a little tool that converts
PNG images of any size into pretty basic ASCII art.

This leverages redscii, a simple image processing library written in Go to downscale, greyscale and finally generate the
ASCII art from an original image.

## Installation

Build it with `go build -o rasciifier`

## Usage
```
rASCIIfier

Usage

  -f string
        Filename of the PNG image to rASCIIfy.
  -w int
        Maximum character width of output ASCII art. (default 300)
```
